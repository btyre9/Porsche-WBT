Image: Art direction for the slide image. Describe subject, composition, mood, and setting.
Video: Art direction for the background video.
Question: (KC / FQ only) The question text
Choice-1 … Choice-4: (KC / FQ only) Answer choices
Correct-Answer: (KC / FQ only) Number of the correct choice
Review-Slide: (KC only) Slide ID to return to on wrong answer
Status: Draft, In Review, or Approved
Notes: Developer notes, feedback text, implementation details — not parsed into course data
