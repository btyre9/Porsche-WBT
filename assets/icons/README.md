# Icons Directory

Place Porsche Design System icons and custom icons here.

## Recommended Icon Assets

- `porsche-logo.svg` - Main Porsche crest logo
- Navigation and UI icons
- Custom course icons

## Naming Convention

- Use lowercase with hyphens: `icon-name.svg`
- Prefix custom icons with `CC[module-number]_icon_` (e.g., `CC01_icon_brand_ambassador.png`)

## Supported Formats

- SVG (preferred for scalability)
- PNG (for raster icons)

## Build Process

Icons in this folder will be automatically copied to `05_output/assets/icons/` during the build process.
